# Vessel Creation Success Screen

## Rule

1. **Direct navigation:** Once a captain clicks "Create Vessel" and the vessel is successfully created, they must go directly to the success screen (no intermediate dialogs that block the flow). The success screen allows the user to share the invite code or proceed to Home.

2. **Theme:** The vessel creation success screen must use the same theme as the Login and Welcome pages (dark maritime: bgDark #0f172a, textOnDark, textMuted, gold accent).

3. **Icon:** Use the Nautical Ops badge (boat icon + "Nautical Ops" text) as on Welcome and Login pages. Do not use the anchor emoji.

4. **Next Steps:** Include in the Next Steps section: "Invite Code is accessible in the settings menu."

## Scope

Applies to CreateVesselScreen (success view).
